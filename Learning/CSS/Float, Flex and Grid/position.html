CSS Position Property Demo

1. static

Im static — this is the default position. It will be positioned according to the normal flow of the document.

2. Relative

Im relative — positioned relative to its normal position. This box is moved down 10px and right 20px from where it would normally be.

3. Absolute

position: absolute — positioned within its nearest positioned ancestor.

(This container is position: relative so the absolute box uses it as its reference.)

4. Fixed

position: fixed — pinned to the viewport. Scroll and I stay put!

5. sticky (header)

I’m position: sticky; top: 0; — I stick to the top as you scroll
